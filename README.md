A simple and user-friendly **Expense Splitter web application** built using **HTML, CSS, and JavaScript**.  
It helps groups split expenses fairly and shows who owes or gets money.


##  Features

- Add multiple group members
- Add expenses with:
  - Expense name
  - Amount
  - Person who paid
  - People involved
- Automatically splits expenses equally
- Calculates net balance for each member
- Shows who owes and who gets money
- Data is saved using **localStorage**
- Works without any backend
